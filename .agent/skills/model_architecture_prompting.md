# Skill: model_architecture_prompting

Use this when the task is about neural network or ML model architecture.

## Principles
- infer main architectural modules from description
- do not require user to list every layer manually
- target 5-8 major blocks
- use compact labels
- avoid low-level implementation details

## Preferred block names
- Input
- Conv Stem
- CNN Blocks
- Transformer Blocks
- Encoder
- Decoder
- Bottleneck
- Fusion
- Cls Head
- Output
